<h2>📖 About the Project</h2>

<p>
This repository contains the implementation of fundamental Data Structure concepts using the C programming language. The programs are designed to help students understand the practical working of various data structures and algorithms.
</p>

<h2>🚀 Topics Covered</h2>

<ul>
<li>Arrays</li>
<li>Array Traversal</li>
<li>Array Insertion</li>
<li>Array Deletion</li>
<li>Linear Search</li>
<li>Binary Search</li>
<li>Bubble Sort</li>
<li>Selection Sort</li>
<li>Insertion Sort</li>
<li>Stack Operations</li>
<li>Queue Operations</li>
<li>Infix Expressions</li>
<li>Prefix Expressions</li>
<li>Postfix Expressions</li>
<li>Expression Conversion & Evaluation</li>
</ul>
